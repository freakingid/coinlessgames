The first proper overhaul pass, mostly about making the game feel like a place you can move around in and control.

- You can now open the pause menu, and Options, from anywhere with **ESC** or the controller's **Start** button. No need to start a game first.
- Added a parallax **starfield** so you can actually feel the ship move against the world.
- Shot and Magnet powerups can now expire by **use** instead of **time**. A new **Difficulty** screen lets you choose, so Triple Shot can last a set number of shots, or the Magnet a set number of pieces collected.
- Rebalanced the gun: the base fire rate is a touch slower, and **Rapid Fire** is now genuinely faster.
- Your **tow capacity grows** as you deliver salvage, up to a new ceiling.
- A full **achievements ladder** (bronze through diamond) plus new lifetime and rotating weekly challenges.
